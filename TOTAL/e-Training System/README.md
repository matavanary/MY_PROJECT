# e-Training System ระบบฝึกอบรม

**สิทธิ์ผู้ใช้งาน**  
  1. ผู้ดูแลระบบ  
    * รายชื่อสมาชิก  
    * จัดการสถาบัน/วิทยากร  
    * จัดการหลักสูตร  
    * จัดการแบบทดสอบ  
    * แผนอบรมประจำปี  
    * จัดการแผน  
    * อนุมัติแผน  
    * ลงทะเบียนอบรม  
    * Training Control Room  
    * จัดการวางบิล  
    * รายงาน
  2. ผู้จัดการ  
    * อนุมัติแผน  
    * รายงาน  
  3. วิทยากร  
    * โปรไฟล์ส่วนตัว  
    * Training Control Room  
    * รายงาน  
  4. พนักงาน  
    * โปรไฟล์ส่วนตัว  
    * ลงทะเบียนอบรม  
  5. Programmer  
    * จัดการสิทธิ์เข้าใช้งาน  
    * จัดการเมนู  
    * และทุกเมนูที่อยู่ในสิทธิ์อื่น ๆ  

**อื่น ๆ**
  1. PHP Language
  1. Javascript Language
  1. CSS Language
  2. Database: MS SQL Server
