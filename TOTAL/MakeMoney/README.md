# MakeMoney ระบบบันทึกเงินเข้า - ออกบัญชี

**สิทธิ์ผู้ใช้งาน**  
  1. ผู้ใช้งาน  
     * ตรวจสอบยอดเงินคงเหลือของทุกบัญชีในหน้า Dashboard 
     * บันทึกรายการเงินเข้า 
     * ตารางบันทึกรายการเงินเข้า
     * บันทึกรายการเงินออก  
     * ตารางบันทึกรายการเงินออก
     * โอนย้ายเงินระหว่างบัญชี
     * เพิ่มผู้ใช้
     * ข้อมูลส่วนตัว
     * เปลี่ยนรหัสผ่าน

**อื่น ๆ**
  1. Javascript Language
  2. HTML Language
  3. CSS Language
  4. PHP Language
  5. Database: MySQLi
